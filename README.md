# fformation

Detects social groups. The group assignment is not implemented yet.

## How do I get set up? ###

    > git clone 
    > mkdir -p fformation/build && cd fformation/build
    > cmake .. && make

## Applications

### fformation-example

An example application of which I classifies groups and prints the corresponding costs.
